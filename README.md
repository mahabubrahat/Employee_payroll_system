💼 Employee Payroll System (C Language)
A beginner-friendly console-based Payroll System written in C. It helps manage employee records, calculate salaries, and store data using both binary and text file handling techniques.

✨ Features
✅ Add, update, search, and delete employee records
💰 Calculate net salary based on basic, allowance, and deduction
🧾 Generate individual salary slips (.txt format)
📊 Export employee data to a CSV file (employees.csv) for Excel/Sheets
💾 Save & load employee data using binary file (employees.dat)
🔐 Password-protected login system (admin123)
📂 Sort employees by name or net salary

🛠️ Technologies Used
Language: C

File Handling:

Binary → .dat

Text → .csv, .txt

IDE: Any C-compatible IDE (e.g., Code::Blocks, Dev C++, VS Code)

📦 Output Files
employees.dat → Binary file storing employee records

employees.csv → Human-readable employee report (Excel/Sheets compatible)

SalarySlip_ID.txt → Auto-generated salary slip per employee

🔐 Default Password
admin123
